## Idea
2 collaborative robots build a tower/wall

## requirements
### base requirements
-	Two robot arms that build a tower between them
  
### extra requirements
-	Bricks with different colors and shapes, and build the tower/wall with different patterns
-	Limited resources, after there are less than 5 bricks, one of the two robots wonder around to retrieve other bricks

-	Different bricks color order
